## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for GitHub API integration

### Local Development

```bash
# Clone the repository
git clone https://github.com/Akshat-A-K/My-Portfolio.git

# Navigate to project directory
cd My-Portfolio

# Open in browser
# Option 1: Double-click index.html
# Option 2: Use a local server (recommended)
python -m http.server 8000
# Then visit http://localhost:8000
```

### Deployment

The portfolio is automatically deployed via GitHub Pages:

1. Push changes to the `main` branch
2. GitHub Actions automatically deploys to GitHub Pages
3. Site is live at `https://akshat-a-k.github.io/My-Portfolio`
